To run the project do the following steps 

1. Install node.js
2. Clone the repository
3. open the correct folder (Shelfmart)
4. open a new terminal
5. Install dependencies (run this code 'npm install')
6. In terminal run 'npm run dev' after installing dependencies
7. Terminal will show your localhost id follow it for view the web page.
8. Everytime to run the project use the commands
       * cd Shelfmart
       * npm run dev
9. If project doesn't work run 'npm install' and run again the above 'npm run dev'
10. Main files:
      *src/App.jsx
      *src/index.css
